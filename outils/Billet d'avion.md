# Billet d'avion moins cher
1. Aller sur [Google Flights](https://www.google.com/travel/flights?hl=fr)
2. Choisir la destination
3. Sélectionner les dates dans le calendrier
4. 1 escale max
5.  Sélectionner le calendrier en bas
